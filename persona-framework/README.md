- `product-manager.md` - Strategic product thinking, roadmap planning
- `product-owner.md` - Backlog management, user stories expert
- `ux-designer.md` - User experience and interface design specialist

### Engineering Team
- `solution-architect.md` - System design and architecture decisions
- `senior-engineer.md` - Code quality, best practices, mentoring
- `frontend-developer.md` - UI implementation and user interactions
- `backend-developer.md` - APIs, business logic, data management
- `devops-engineer.md` - Infrastructure, CI/CD, deployment automation

### Specialized Roles
